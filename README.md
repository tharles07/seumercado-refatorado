# Sistema de Mercado 🛒

## 📌 Sobre o projeto
Sistema completo para gerenciamento de mercado com cadastro de:
- Clientes
- Produtos
- Fornecedores
- Usuários

## 🚀 Tecnologias
- Java
- JPA / Hibernate
- MySQL
- Swing

## 📷 Funcionalidades
- Login com validação
- CRUD completo
- Relatórios

## ▶️ Como rodar
1. Clone o projeto
2. Configure o banco
3. Execute no NetBeans

## 📈 Melhorias futuras
- API REST com Spring Boot
- Interface web
